# Kurum Arçza Takip Sistemi 
 
Bu proje, kurumsal sistemlerde oluüan arçzalarçn kaydçnç tutmak, sçnçflandçrmak ve yînetmek amacçyla geliütirilmiütir. Django ve React teknolojileri kullançlmçütçr. 
 
## Kullançlan Teknolojiler 
- Django Rest Framework 
- React.js + Tailwind CSS 
- PostgreSQL 
- JWT Authentication 
- NLP ile otomatik kategori ve încelik atama
